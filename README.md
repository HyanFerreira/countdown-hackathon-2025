# CountDown - Hackathon 2025 IFSP Caraguatatuba

⏳ Aplicação de contagem regressiva e contagem progressiva desenvolvida para o **Hackathon 2025** do **IFSP - Campus Caraguatatuba**.

## 💡 Sobre o Projeto

O **CountDown** é um sistema web que exibe:
- ⏱️ Uma contagem progressiva até o início do evento.
- ⏳ Uma contagem regressiva após o início do evento.
- 🔴 Alertas visuais e sonoros em momentos críticos da contagem (1h30 antes e 1 minuto antes do término).

O projeto utiliza integração com o **Firebase Realtime Database** para manter a sincronização dos dados entre diferentes dispositivos.

---

## 🚀 Tecnologias Utilizadas
- **HTML5**
- **CSS3**
- **JavaScript (Vanilla)**
- **Firebase Realtime Database**

---

## 🔥 Funcionalidades
- Contagem progressiva até a data/hora de início do evento.
- Contagem regressiva após o início.
- Sincronização em tempo real via Firebase.
- Alertas sonoros e visuais em tempos críticos.
- Animação de destaque quando o alerta é disparado.
